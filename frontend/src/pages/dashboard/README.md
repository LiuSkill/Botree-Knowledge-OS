# Dashboard Module

## 功能

首页工作台用于展示欢迎区、核心指标、快捷入口、最近上传文档、最近 AI 问答和知识分类统计。

## 调用关系

路由模块将 `/dashboard` 指向 `DashboardPage.vue`。页面调用 `getDashboardStats()` 读取 `/api/system/dashboard`，并使用 `ROUTE_PATHS` 统一处理跳转路径。

## 输入

- 后端工作台统计响应：项目数量、文档数量、知识条目数、AI 回答次数和待办审核数。
- 后端最近数据：最近文档、最近 AI 提问、知识分类统计。
- 当前登录用户信息：用于欢迎区展示姓名。

## 输出

- 首页工作台页面。
- 知识中心、项目中心、企业知识问答、项目知识问答和文档详情入口。
- 后端为空时展示空状态，不补充前端样例数据。
